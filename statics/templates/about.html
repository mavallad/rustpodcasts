About

Welcome to rustpodcasts, your ultimate resource for Rust programming language podcasts.

Our mission is to provide a curated collection of insightful podcasts that delve into the world of Rust, catering to beginners and experienced developers alike.
Immerse yourself in the Rust community, stay updated on the latest trends, and level up your programming skills.

Help us grow

If you have a favorite podcast to recommend, let us know at rustpodcasts@gmail.com
Spread the word by sharing our website with fellow Rustaceans.

Contact us

  • email: rustpodcasts@gmail.com
  • X, Twitter: @rustpodcasts

Code

This website is open source and available on github at https://github.com/mavallad/rustpodcasts.

The following technology has been used:

  • Shuttle: deployment infrastructure for Rust (https://www.shuttle.rs). Thanks a lot for the great work and the free tier!
  • PostgreSQL: relational database system (https://www.postgresql.org)
  • Actix-web: web framework for Rust (https://actix.rs)
Thanks to all the authors and contributors of these projects and the rest of the libraries used to build this website!

If you have improvements to suggest, please open a pull request on github or send an email.
